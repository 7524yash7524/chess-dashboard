# ♟️ Chess Dashboard

An interactive analytics dashboard for Chess.com players built with Streamlit and Plotly.

## 🔴 Live Demo
[Click here to open the dashboard](https://chess-dashboard-laxmedzvjilbmgpubrad4v.streamlit.app/)

## 📊 Features
- Overview — total games, win rate, W/L/D breakdown
- Rating over time by time control
- Performance by time control (Rapid, Blitz, Bullet, Daily)
- Openings analysis — best and worst openings by win rate
- Opponent analysis — win rate vs stronger/weaker opponents
- Accuracy trend over time
- Time patterns — games per month, best and worst month
- Sidebar filter by time control
- Upload your own CSV to see your stats

## 🚀 How to Use
**Option 1 — Use the live app**
Upload your own `chess_games.csv` generated by the fetch script.

**Option 2 — Run locally**
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📁 Generate Your Own Data
1. Open `fetch_games.ipynb`
2. Change `USERNAME` to your Chess.com username
3. Run all cells — saves `chess_games.csv`
4. Upload to the dashboard

## 🛠️ Tech Stack
- Python · Streamlit · Pandas · Plotly · Chess.com Public API
